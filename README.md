# Homebrew Tap for dastrobu

This is a [Homebrew](https://brew.sh) tap for my projects.

## Installation

```bash
brew tap dastrobu/tap
```

## Available Formulae

### apple-mail-mcp

MCP server providing programmatic access to macOS Mail.app.

```bash
brew install apple-mail-mcp
```

See the [apple-mail-mcp repository](https://github.com/dastrobu/apple-mail-mcp) for more information.

## Updating

```bash
brew update
brew upgrade apple-mail-mcp
```

## Uninstalling

```bash
brew uninstall apple-mail-mcp
brew untap dastrobu/tap
```

## About

This tap is automatically updated by [GoReleaser](https://goreleaser.com/) when new releases are published.

The formula files are located in the `Formula/` directory.
